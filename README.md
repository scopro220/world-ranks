# World Ranks Web Application

## Next JS / CSS Modules

![Next JS](https://img.shields.io/badge/Next-black?style=flat&logo=next.js&logoColor=white)

Website: [World Ranks](https://world-ranks-seven-ashen.vercel.app/)

Hosted on [Vercel](https://vercel.com)

## API Reference

- [Rest Countries](https://restcountries.com/)
- Get information about countries via a RESTful API
- Version used: 3.1

## Features

- Mobile Responsive
- Search for countries by Name, Region or SubRegion
- Light/Dark Theme
- Sortable by Name, Population and Area
- Data on individual country pages
